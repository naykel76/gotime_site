Installation

Get started with our framework by installing it via npm or using a CDN link.

Install via npm

The recommended way to install the framework is through npm. This allows you to easily manage dependencies and stay up to date with the latest releases.

npm install framework-name
# or
yarn add framework-name
Note
Make sure you have Node.js installed on your system before running the npm install command.

Using CDN

For quick prototyping or testing, you can include the framework directly via CDN without any build process.

<link href="https://cdn.example.com/framework.css" rel="stylesheet">

Basic Setup

After installation, import the framework in your main JavaScript or CSS file to start using it in your project.

import 'framework-name/dist/framework.css';
import { Component } from 'framework-name';

Configuration

Create a configuration file at the root of your project to customize the framework settings according to your needs.

module.exports = {
theme: {
colors: {
primary: '#38bdf8',
secondary: '#818cf8'
}
}
}
